みずのほし について

-頭の中だけで上映される映画-


映画には必ず映像があります。

様々な表情や景色を切り取り繋げる映像は、映画にとって欠かせない存在です。


しかし『みずのほし』には、”映像”がありません。

その代わりに、私たちは”音楽・脚本・イラスト”の要素で作品を形作りました。

そして作品を完成させる為にはもう一つ必要な要素があります。

それは想像することで、頭に浮かぶ”物語の景色”です。

”音楽・脚本・イラスト”とあなただけの”景色”

これらの要素が合わさることで、この作品は映画として完成します。


『みずのほし』が新しい映画体験となること、

そして受け取ってくださった方と、見えないけれど、

確かな繋がりを得られたらーーそう願って、制作をしました。


イラストを見て、脚本を読み、音楽を聴き、目を瞑り、

あなただけの『みずのほし』を上映してください。

最後まで上映が止まらないことを願っています。
